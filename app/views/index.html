Git & Heroku Deploy my prototype

Setting up Git:

  • Create a new Git repository
  • Download nhs prototype kit and drop into a new folder
  • Open your prototype directory in the terminal (right click, open in terminal)
  • In terminal:
    • git status - show all new/changed files (red)
    • git add filename - add new file (first time you can do git add * to add everything)
    • git commit -m "message here" - commit new files to be pushed
    • git remote add origin https//new-github-repo.url - connect your folder to the new github repo
    • git push -u origin master - push the commits to the new repo's master branch

    That's it for the Git setup, now to link up to Heroku

    Setting up Heroku:

    • Login or make a Heroku account
    • login to the dashboard and create a new app (this will be the name in the URL)
    • Deploy tab: Deployment method, choose git and search for your new repo to connect to, then below choose a branch and select enable automatic deployment
    • On the settings tab, reveal Config vars and set username and password as key/values: PROTOTYPE_USERNAME: / PROTOTYPE_PASSWORD
    • Go back to deploy tab, under Manual deploy, hit deploy branch. Once inital build is deployed, go to the url provided and you should be prompted for the username and password for your new heroku prototype!
